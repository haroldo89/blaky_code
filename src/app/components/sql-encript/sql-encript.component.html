CIFRAR Y DESCIFRAR EN SQL SERVER

¿Cómo cifro y descifro cualquier cadena usando sql server?



Una frase de contraseña para generar una clave simétrica.


Una variable de tipo nvarchar, char, varchar, binary, varbinary o nchar que contiene una frase de contraseña para generar una clave simétrica.


El texto sin cifrar a cifrar.


Una variable de tipo nvarchar, char, varchar, binary, varbinary o nchar que contiene el texto en claro. El tamaño máximo es de 8,000 bytes. 

    
    DECLARE @encrypt  VARBINARY(200);
    SELECT @encrypt = EncryptByPassPhrase('key', 'value'));
    SELECT @encrypt;
    SELECT CONVERT(VARCHAR(100), DecryptByPassPhrase('key', @encrypt))
